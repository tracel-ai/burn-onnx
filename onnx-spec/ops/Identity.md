# Identity

First introduced in opset **1**

All versions: 1, 13, 14, 16, 19, 21, 23, 24

## Description

Identity operator

## Inputs (1 - 1)

- **input** (V): Input tensor

## Outputs (1 - 1)

- **output** (V): Tensor to copy input into.
